A little python site

1) For Linux/MacOSX user

Please execute

```
export PYTHONPATH=$PYTHONPATH:.
python bin/app.py
```

then visit <http://localhost:8080> to browse the site!

1) For Windows user

Please execute

```
$env:PYTHONPATH = "$env:PYTHONPATH;."
python bin/app.py
```

then visit <http://localhost:8080> to browse the site!
